# Websites

1.# She Can Foundation Website

## Overview

This project is a responsive website developed for **She Can Foundation**, a non-profit organization dedicated to empowering women through education, skill development, mentorship, and community support.

The website provides visitors with information about the foundation's mission, ongoing initiatives, events, volunteer opportunities, and ways to contribute. It is designed with a modern, user-friendly interface to improve accessibility and engagement.

## Features

- Responsive and mobile-friendly design
- Modern and intuitive user interface
- About the Foundation section
- Mission & Vision
- Programs and Initiatives
- Events and Campaigns
- Volunteer Registration
- Contact Form
- Social Media Integration
- Smooth Navigation

## Tech Stack

- React.js
- Vite
- Tailwind CSS
- JavaScript
- HTML5
- CSS3

## Purpose

The primary goal of this project is to strengthen the digital presence of She Can Foundation by providing an informative, accessible, and visually appealing platform that connects the organization with volunteers, donors, and the community.
